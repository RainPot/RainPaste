# RainPaste 命令面板改造设计文档

**日期：** 2026-03-12

## 目标

将当前偏“大窗口管理器”的主界面收缩成更接近 `Spotlight / Raycast` 的浮层式命令面板。面板仅保留历史搜索与快速复制能力，暂停监听和清空历史等管理动作统一回归状态栏菜单。

## 范围

### 本次包含

- 全局快捷键改为 `⌘⇧V`
- 主窗口改为无标题栏、无红黄绿按钮的纯面板
- 支持 `Esc` 关闭面板
- 删除右侧控制面板
- 主界面改为单栏、尺寸缩小
- 底部保留精简状态信息
- 重新打包 `.app`

### 本次不包含

- 快捷键自定义编辑
- 左右 `Shift` 物理键区分
- 新的设置页或多窗口
- 自定义应用图标资源

## 交互设计

### 唤起与关闭

- 使用 `⌘⇧V` 唤起窗口。
- 唤起后自动激活应用并聚焦搜索框。
- 按 `Esc` 关闭当前面板。
- 点击某条历史后写回剪贴板，并在“复制后自动关闭窗口”开启时关闭面板。

### 面板布局

- 顶部：应用标题、快捷键标签、历史数量
- 中部：搜索框与历史列表
- 底部：监听状态、提示文案

### 菜单栏职责

- 打开主面板
- 暂停或恢复监听
- 清空历史
- 退出应用

## 技术方案

1. `MainWindowCoordinator`
   - 使用更像浮层的 `NSPanel` 风格窗口配置。
   - 隐藏标题栏与标准窗口按钮。
   - 调整窗口尺寸与最小尺寸。

2. `MainWindowView`
   - 删除右侧控制面板。
   - 采用单栏紧凑布局。
   - 保留搜索框、列表和底部状态条。
   - 通过回调支持 `Esc` 关闭。

3. `AppViewModel`
   - 增加关闭动作分发。
   - 提供更适合单栏面板展示的状态文本。

4. `GlobalHotKeyService`
   - 默认快捷键改为 `⌘⇧V`。
   - 测试更新为新的展示文案。

## 风险与说明

- 基于当前全局热键实现，仅能注册 `Shift` 修饰键，不能区分左 `Shift` 与右 `Shift`。
- `Esc` 关闭需要确保搜索框聚焦时仍能生效，因此会在窗口层和视图层双重兜底。

## 测试策略

- 单元测试验证默认快捷键展示为 `⌘⇧V`
- 单元测试验证视图模型的关闭动作可被触发
- 构建验证确保 SwiftUI 视图与窗口桥接正常编译
